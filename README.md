# node-project-template
node project template

## 安装node
1. 安装nvm

linux/mac安装
```
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
source ~/.bashrc
nvm install 10
```
windows安装

从```https://nodejs.org/zh-cn/```下载npm安装包，直接安装


2.安装npm源管理工具nrm
```
npm install -g nrm
```

3.切换淘宝源
```
nrm use taobao
```
